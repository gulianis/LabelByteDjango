Instructions:



  1. Create an account and login

  2. Compress folder of unlabeled images into a zip file

  3. Click label tab and upload zip file
    (label tab cannot be accessed on mobile web)

  4. Download the app and login with the same account

  5. Click on zip name and image name of image you would like to label

  6. Instructions in top right corner has information on how to label

  7. After you finish labeling an image press save

  8. Go back to label tab on website and click download.
    Saved labels will be listed in a text file.
